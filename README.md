# Vectorium
Una aplicación sencilla diseñada con la ayuda de la herramienta Codex para un proyecto escolar, que ayuda principalmente a realizar derivadas e integrales de vectores. También cuenta con algunas otras funciones relacionadas a cálculo vectorial.
